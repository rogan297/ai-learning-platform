## Technology Architecture

The platform follows cloud-native principles and is fully built on
Kubernetes, using Docker images for application containerization.

### Core Components

-   **KAgent** for intelligent agent orchestration and lifecycle
    management\
-   **KMCP** for implementing Model Context Protocol (MCP) servers\
-   **AgentGateway** for AI security and access control\
-   **Keycloak** for identity and access management\
-   **OpenTelemetry Collector** for observability, tracing, and
    monitoring\
-   **PostgreSQL** as a robust and scalable relational database
-   **LangGraph** for creating a custom agent workflows
### Architectural Capabilities

-   Horizontal scalability\
-   High availability\
-   Centralized security management\
-   Full observability of agent behavior\
-   Modular and evolvable system design
